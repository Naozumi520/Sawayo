# Sawayo
Sawayo, a discord bot for osu!

## Set-up
Clone the git project:
```bash
$ git clone https://github.com/Naozumi520/Sawayo.git
```
This project uses node and npm. Use the following command if you don't have them locally installed.
```bash
$ cd Sawayo/
$ npm install
```
To start the bot:
```bash
$ node index
```
